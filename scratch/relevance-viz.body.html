Four honest stops, never a score. The matrix above explains what each verdict turns on.

Every value shown — verdicts, footprints, decisiveness — is computed by the real ASSAY engine (compile → score → materialise, engine ) and pre-rendered; the page adds no numbers of its own. Operation Sandpiper is an exploratory vignette, not Meridian — a scratch demonstrator built to show the shape, not a canonical fixture. Relevance is computed two ways that agree: read-set (what the scorer reads) and removal (what changes a verdict when withheld).
